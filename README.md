# Catalogo sulle opere di Michelangelo
Materiale per la demo di un sito che esplora un catalogo sulle opere di Michelangelo:
- pagina home (index.html)
- pagina catalogo opere (catalogo.html)
- pagina singolo oggetto del catalogo (list-item.html)
- pagina su Bologna, sottosezione di Itinerari (mappa.html)
- pagina sulla vita (la-vita.html)
- pagina Web Project Plan (web-project-plan.html)
- pagina con i dati (dati.html)
- file CSV con i dati (dati.csv)
- cartella con tutte le immagini utilizzate (img)
- cartella con il file CSS utilizzato (style/style-michelangelo.css)
- cartella con gli script java utilizzati (js/main.js)
